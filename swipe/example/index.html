Swipe.js

JavaScript plugin for creating and analyzing swipe events. Compatible with desktop and mobile browsers, support touch and mouse events.

Usage

var swipe = new Swipe(element);

swipe.bind({

 start: function() { ... }

 move: function() { ... }

 end: function() { ... }

});

Browser support

  • Chrome
  • Firefox
  • Safari
  • IE 9+
  • Opera 7+

Mobile support

  • Android 2+
  • iOS 6+
  • Windows Phone 8+
